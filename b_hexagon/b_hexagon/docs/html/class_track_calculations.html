Detailed Description

Pure stateless calculation utilities for TrackData timing metrics.

Centralizes arithmetic logic to keep the application service (orchestrator) lightweight and focused on workflow. Facilitates targeted unit testing of business rules. All returned / consumed time values are expressed as nanoseconds since epoch, represented as double for convenience.
