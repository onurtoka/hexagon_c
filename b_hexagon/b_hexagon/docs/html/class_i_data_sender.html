Detailed Description

Abstraction representing a sink/egress for enriched track data.

Implementations (adapters) may forward data via messaging middleware (ZeroMQ PUB), write to stdout, persist to storage, or publish over a network protocol. The domain layer depends only on this interface (Dependency Inversion Principle), enabling technology-agnostic business logic.
