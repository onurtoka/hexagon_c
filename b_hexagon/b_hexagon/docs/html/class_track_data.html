Detailed Description

Input domain value object carrying raw track kinematic & timing attributes.

Acts as a Data Transfer Object (DTO) at the domain boundary. It is populated by incoming adapters (IDataReceiver implementations) from external representations (e.g., JSON, console). All temporal values are expressed as nanoseconds since UNIX epoch for uniform precision.
