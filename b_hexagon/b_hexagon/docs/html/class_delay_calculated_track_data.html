Detailed Description

Output domain value object aggregating original TrackData fields plus derived delay metrics.

Produced by the application service after invoking pure calculation routines. Adapters implementing IDataSender serialize this DTO outward (e.g., JSON via ZeroMQ). All timing values are in nanoseconds.
