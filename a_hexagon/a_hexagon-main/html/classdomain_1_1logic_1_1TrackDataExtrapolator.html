Detailed Description

Main business logic class for track data extrapolation.

This class implements the core extrapolation algorithm that converts incoming track data from 5Hz frequency to 200Hz frequency using constant velocity model. It follows the Hexagonal Architecture pattern by implementing the TrackDataIncomingPort interface.

Processes incoming track data and forwards extrapolated results.

This method receives track data at 5Hz, extrapolates it to 200Hz for a 125ms window (25 data points), and forwards the results through the configured outgoing port.
