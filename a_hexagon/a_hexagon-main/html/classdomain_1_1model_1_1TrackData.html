Detailed Description

Input data model for track information at 5Hz frequency.

This class represents incoming track data containing position, velocity, and timing information in ECEF (Earth-Centered, Earth-Fixed) coordinate system. Data is typically received from radar systems, sensors, or other tracking devices at 5Hz frequency.
